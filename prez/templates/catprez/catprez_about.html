About CatPrez

CatPrez is a read-only web delivery system - web pages and API - for Data Catalog Vocabulary (DCAT) catalogue information. It complies with Content Negotiation by Profile.

Introduction

This is an instance of CatPrez. It is a part of a Linked Data presentation API called Prez. This instance is the first demo version of Prez.
